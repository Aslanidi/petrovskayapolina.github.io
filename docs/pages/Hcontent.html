Контент-теги
Контент-теги – это неофициальное название группы тегов, при помощи которых информация размещается и позиционируется относительно друг друга в HTML документе.
Заголовки
Для создания заголовков в HTML документе необходимо использовать парный тег <hX>. Вместо Х располагается значение от 1 до 6 – это называется градацией заголовков.
Пример градации заголовков, от h1 до h6

По правилам вёрстки, самый крупный заголовок <h1> настоятельно рекомендуется использовать только один раз на странице, поскольку благодаря именно ему осуществляется поиск по ключевым словам в поисковых системах.


Теги заголовков с остальными градациями можно использовать исходя из смыслового деления информации.

Параграфы
Следующим одним из самых важных контент-тегов является тег <p>, при помощи которого текст в HTML-документе разделяется на абзацы. Абзац всегда создается с новой строки, вместе с небольшим отступом перед и после текста.
Пример использования тега <p>
Особенностью тега <p> является то, что он не «чувствителен» к переносам строки, а так же множественным пробелам внутри текста.
Стихотворенние и текст с большим количеством пробелов всё равно отображаются в одну строку
Так же параграфы между собой можно разделять при помощи горизонтальной линейки. Для этого используется одиночный тег <hr>.
Пример использования тега <hr>
Разрыв внутри параграфа («насильны» перенос строки) осуществляется при помощи одиночного тега <br>.
Пример использования тега <br>
Списки

HTMl документ поддерживает три типа списков:


  • 1. Маркированный
  • 2. Нумерованный
  • 3. Описание

Тег <ul> позволяет создать маркированный список. По умолчанию все пункты списка будут промаркированы с помощью небольшой черной точки. Изменить символ можно при помощи стилей.


Тег <ul> создает непосредственно «тело» списка, для того, чтобы добавить позиции, необходимо использовать парный тег <li>, внутри которого следует размещать непосредственно, содержимое списка.

Пример маркированного списка <ul>
Те же самые правила работают и с нумерованным списком. По умолчанию пункты списка маркируются арабскими цифрами.
Пример нумерованного списка <ul>

«Описание» формирует список, в котором к каждому объекту будет дано описание.


«Тело» списка формирует парный тег <dl>. Парный тег <dt> заключает в себя название объекта, а <dd> - описание.
