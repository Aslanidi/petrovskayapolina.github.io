Программное обеспечение

Одним из главных преимуществ frontend разработки (да и всего программирования в частности) является то, что HTML код можно написать в самом обычном блокноте (электронном, конечно же).


Но для того, чтобы упростить и в разы ускорить данный процесс, были разработаны различные программы для занятий программированием. Между собой они отличаются только представленным функционалом.

Редакторы кода
  • Visual Studio Code — редактор исходного кода, разработанный Microsoft для Windows, Linux и macOS. Позиционируется как «лёгкий» редактор кода для кроссплатформенной разработки веб- и облачных приложений. Поддерживает множество различных плагинов, упрощающих и ускоряющих работу.

  • Sublime Text — проприетарный текстовый редактор. Поддерживает плагины на языке программирования Python. Так же как и VSC поддерживает множество различных плагинов.

  • Atom – редактор кода, разработанный авторами Github, поддерживает прямую связь со своим «родителем».

  • Dreamweaver — визуальный HTML-редактор компании Adobe. Позволяет отслеживать результат в режиме реального времени, не переключаясь между вкладками.
Графические редакторы
  • Figma — онлайн-сервис для разработки интерфейсов и прототипирования с возможностью организации совместной работы в режиме реального времени.

  • Adobe Photoshop – современный, наиболее распространенный графический редактор, позволяющий создавать макеты сайтов.

  • Avocode – программа для просмотра макетов. Позволяет импортировать встроенные в макет элементы и изображения. Отображает CSS код для отдельных элементов сайта.
