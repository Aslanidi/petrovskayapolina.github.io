Что такое HTML

HTML – это язык гипертекстовой разметки документов, который применяется для создания веб-страниц. Образно выражаясь, HTML можно сравнить со «скелетом», каждая «косточка» которого является отдельным элементом, формирующим общую структуру.


В основе любого HTML документа лежит система тегов – парных меток, которые являются основой кода любой веб-страницы. Данные теги могут описывать совершенно различные элементы, такие как информационные блоки, заголовки, таблицы, списки и многое другое.

Пример структуры тега

Важно помнить, что HTML документ является не более чем «скелетом», в нем невозможно настроить стили отображения – для этого существует отдельный язык под названием CSS.


Кроме того, важной особенностью HTML документов является принцип вложенности, т.е. возможности располагать элементы по принципу матрешки – один внутри другого. Это позволяет сформировать сложную многоуровневую структуру веб-страницы.

Плюсы и минусы HTML
К плюсам HTML можно отнести
  • •Широкое распространение по всему миру;
  • •Возможность просмотра в любом браузере;
  • •Бесплатный открытый доступ;
  • •Интуитивно понятный синтаксис;
  • •Наличие официальных стандартов.
Однако, существуют так же и минусы:
  • • Отсутствие динамических элементов (приходится подключать сторонние языки, например, JavaScript);
  • • Каждую веб-страницу приходится создавать отдельно, даже если она использует одни и те же элементы;
  • • В старых бразуерах могут не поддерживаться отдельные теги.
Связь с другими языками

Если HTML является «скелетом», то ему так же необходима «внешняя оболочка» и «двигательный аппарат».


Функцию «внешней оболочки» выполняет язык CSS, который отвечает за стили отображения контента: фон, цвета, интервалы, базовая анимация.


Функцию «двигательного аппарата», в свою очередь, берет на себя язык JavaScript, который позволяет добавить на веб-страницу динамические элементы: слайдеры, ползунки, всплывающие окна, продвинутую анимацию.

Структура HTML
На самом деле, для того, чтобы создать веб-страницу нужно приложить не так уж много усилий. Например, так выглядит простейший HTML документ.

Простейший (начальный) HTML документ состоит из одного одиночного тега и четырех парных. Именно эта структура показана на вышестоящем изображении. Рассмотрим подробнее каждый из представленных тегов:


!DOCTYPE html – является так называемым сигнальным тегом – он сообщает браузеру о том, какого типа страницу он открывает.


Html – данный парный тег является «контейнером», в котором располагается всё содержимое веб-страницы.


Head - является парным тегом, в котором пишется обязательный тег title, внутри которого указывается название страницы. Именно оно отображается в выдаче поисковика и на вкладке браузера. Так же внутри парного тега head располагаются различные мета-теги, а также подключаются различные стили (например, CSS), а также скрипты (например, JavaScript).


Body – данный парный тег так же является контейнером, но уже для видимого, функционального содержимого сайта. Если в теге head отображается техническая информация (кодировка, название, подключение внешних элементов), то внутри этого тега располагается непосредственное наполнение веб-страницы, которое может быть выражено в виде различных элементов.


Более подробно об остальных тегах можно прочитать в соответствующем разделе.
