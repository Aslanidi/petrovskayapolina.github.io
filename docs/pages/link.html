Ссылки

Немаловажную роль в формировании HTML документов играют ссылки. Они достаточно просты по своей структуре – для создания ссылки необходимо использовать парный тег <а> с атрибутов href=””.

link text
Атрибуты таргета

Существует отдельная группа атрибутов, которая регулирует способ открытия ссылки. Рассмотрим их поподробнее:

  • _self – этот атрибут описывает состояние всех ссылок по умолчанию. Новая страница открывается в той же вкладке, что и предыдущая;
  • _blank – этот атрибут открывает новую страницу в отдельной вкладке;
  • _parent – этот атрибут открывает новую страницу непосредственно на «родительской», в виде отдельного окна.

ЗАКЛАДКИ

В HTML документах существует возможность создавать ссылки-«закладки». Такие ссылки не перенаправляют пользователя на другую страницу, а перемещают его внутри родительской к определенному маячку.

Для того, чтобы создать такую структуру, необходимо присвоить определенному элементу атрибут id, после чего указать его в качестве источника внутри элемента ссылки.
