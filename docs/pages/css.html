Что такое CSS

CSS (Cascading Style Sheets букв. «Каскадные таблицы стилей») – это один из формальных языков программирование, целью которого является описание оформления внешнего вида HTML документа.


Образно говоря, HTML документ представляет собой «скелет», а CSS, в свою очередь – «мышцы» или даже «внешнюю оболочку».

На данной картинке изображен пример сочетания CSS (сверху) и HTML (снизу) кода.


Достоинства
  • • Быстрота разработки;
  • • Возможность повторного использования элементов;
  • • Гибкость и удобство редактирования;
  • • Упрощение кода путем избегания повторов;
  • • Ускоряет время загрузки веб-страницы;
  • • Увеличивает количество визуальных решений;
  • • Гибкая адаптация для мобильных устройств.
Подключение CSS
Пример ссылки на подключение CSS-документа.

Подключить CSS к HTML можно тремя способами:


  • • Внутри тега с помощью атрибута style. При этом нет нужды указывать селектор;
  • • Добавить тег style с атрибутом type="text/css";
  • • Подключить внешнюю таблицу стилей: link rel="stylesheet" href="путь до style.css" type="text/css".

Третий способ является наиболее распространенным, поскольку он позволяет упорядочить «рабочее место», тем самым значительно упрощая и ускоряя работу над кодом.

Cинтаксис

В отличии от HTML, CSS имеет крайне простую структуру записи: она состоит всего из двух элементов: селектора и блока объявлений.

Пример ссылки на подключение CSS-документа.

Селектор определяет, к какому элементу HTML документа будут применены свойства, указанные в CSS документе.


В таблице каскадных стилей можно обращаться как к типу элемента («div», «p», «a» и т.д.), так и к идентификатору, который является ссылкой на CSS стиль. (div class=”title”).


Блок объявлений состоит из пары субэлементов, которые записываются через двоеточие: «свойство: значение». Свойство задает тот или иной аспект (цвет, размер, отступ и т.д.), а значение задает характеристики данного параметра (оттенок, размер в пикселях, процентах и т.д.).
