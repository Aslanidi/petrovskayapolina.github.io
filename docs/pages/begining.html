С чего начать?

Front-end – это один из самых легких и доступных разделов программирования для начинающих. Научится верстке может любой желающий, даже тот, кто до этого вообще не сталкивался с архитектурой программного обеспечения.


На данный момент существует множество различных ресурсов, благодаря которым можно бесплатно и продуктивно изучать основы frontend.

Полезные сайты

Htmlbook – это сайт-справочник, посвященный HTML и CSS. Его стоит использовать как некую энциклопедию или «настольную книгу».


W3School – сайт с интерактивным последовательным изучением HTML, CSS, JavaScript. Удобен тем, что каждое представленное решение можно посмотреть в отдельном окне, где одновременно будет отображаться исходный код и получаемый результат. В конце каждой темы есть тест для закрепления знаний.


Free Code Camp – наиболее популярный ресурс для изучения основ Frontend. Сайт представляет собой полностью интерактивный курс, во время которого пользователь выполняет задания, путем написания кода прямо на веб-странице.

Курсы на Youtube
В социальной сети Youtube размещено немало различных бесплатных курсов, посвященных Frontend разработке. Наиболее полезным, на мой взгляд, является канал «Фрилансер по жизни». Автор данного канала сформировал несколько тематических плейлистов, в которых рассматриваются те или иные аспекты вёрстки.
  • Основы front-end
  • Мастер-классы по созданию различных элементов
  • Вёрстка готовых примеров
