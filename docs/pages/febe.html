Что такое Frontend и Backend

Все веб-страницы, которые можно найти во всемирные паутины созданы при помощи технологий Frontend и Backend. И то, и другое является вариантом архитектуры программного обеспечения.

Данные термины появились в программной инженерии, вследствие разделения задач между программистами.


Frontend
это – внешняя составляющая любой веб-страницы, то есть, всё то, что видит пользователь, открывая сайт. Сюда относится пользовательский интерфейс, а также его функционального наполнение.

В любом браузере можно открыть исходный код (зачастую для этого необходимо нажать на клавишу F12) после чего откроется панель разработчика, в которой можно будет увидеть frontend структуру веб-страницы.

К Frontend разработке относят:


  1. HTML – это стандартизированный «язык» разметки документов во всемирной паутине, на котором написаны большинство сайтов. Принцип работы HTML состоит в том, что браузеры интерпретируют исходный код, отображая веб-страницу на экране монитора или мобильного устройства.
  2. CSS – это язык стилизации HTML. Проще говоря, HTML – это содержание, а CSS – стиль отображения существующего контента. При помощи этого языка задаются различные параметры, влияющие на отображение, положение, размер многих элементов, таких как текст, изображения, ссылки и т.д.
  3. JavaScript – это язык, благодаря которому осуществляется интерактивное взаимодействие между веб-страницей и пользователем. JavaSсript отвечает за анимацию, формы обратной связи, загрузку данных и многое другое.

Backend
это программно-аппаратная часть веб-страниц. Сюда входят те средства, при помощи которых происходит реализация скрытых процессов веб-сайта.


Backend реализует связь между пользователем и сервером.


Технологии backend реализуются при помощи других универсальных языков программирования: PHP, Ruby, Python, Java.
